# AI UX-UI Design Principles

Core design instructions and parameters utilized during the Spoolman Dark UI redesign.

## 1. Goal
Upgrade the visual style of SpoolmanDB Community database explorer to match the Maker Tool / 3D printing community aesthetic, specifically aligned with the official Spoolman web client's dark interface.

## 2. Technical Stack Rules
*   **Vanilla CSS:** Pure custom CSS layout styling, no external utility frameworks.
*   **HTML5 Semantics:** Retain existing grid and layouts on tabs other than Home, keeping DOM nodes identical to avoid functionality breakage.
*   **Interactive ECharts:** Deep grey backgrounds with neon gold/orange colors, custom rounded scatter swatches, and high-performance rendering.

## 3. Colors & Theme
*   **Deep Dark Space:** `#0A0A0F`
*   **Maker Cards:** `#18181B`
*   **Gold Accents:** `#F59E0B`
*   **Warm Highlights:** `#F97316`
